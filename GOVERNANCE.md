# GOVERNANCE.md
Open Source Governance & Acceptable Use Policy

（嚴格版｜適用 AI / Agent / Automation 專案）

## 1. 專案性質聲明 / Project Nature
- 繁體中文：本專案為開源軟體專案，僅提供原始碼與技術示範用途。專案創建者與維護者不對使用者的實際部署、修改方式、用途或衍生行為進行任何形式的審核、監管或背書。
- English: This project is an open-source software project provided for source code access and technical demonstration purposes only. The project creator and maintainers do not review, monitor, endorse, or control how the software is deployed, modified, or used.

## 2. 惡意行為與權限終止 / Malicious Use & Termination of Access
- 繁體中文：若任何人在專案相關空間（repository、issue、PR、release、mirror 等）加入、散布或連結惡意程式碼／外掛／模組／執行檔，或實作可危害設備、系統、網路或資料安全之功能，或進行未授權資料蒐集、監控、API 濫用或隱私洩漏，維護者有權立即終止其存取、貢獻或互動權限，無需事前通知。
- English: If any individual introduces malicious code/plugins/modules/executables or implements functionality that compromises system/network/data security, or engages in unauthorized data collection, surveillance, API abuse, or privacy violations, maintainers may immediately revoke access, contribution rights, and participation without prior notice.

## 3. 非自用、商業與組織使用 / Non-Personal, Commercial, or Organizational Use
- 繁體中文：若用於非個人自用（商業、公司/企業/政府/學校/研究機構/團體，或對外服務/產品/平台/AI 系統），使用者必須自行確保符合法律、法規、組織或國家要求並取得授權。
- English: For non-personal use (commercial, organizational, or as part of external services/products/platforms/AI systems), users must ensure compliance with all applicable laws, regulations, organizational rules, and jurisdictions, and obtain proper authorization.

## 4. 禁止用途 / Prohibited Use
- 繁體中文：不得用於違法、詐欺、造假、危害研究、侵害個人/企業/國家/公共基礎設施/AI 系統的行為，或攻擊性、監控性、破壞性、未授權研究。
- English: Do not use for illegal, fraudulent, deceptive, harmful, or rights-infringing purposes, or for offensive/surveillance/destructive/unauthorized research activities.

## 5. AI / Agent / Automation 使用限制 / Usage Restrictions
- 繁體中文：本專案不具備道德、法律或社會判斷能力；不得賦予 AI 超出法律、組織或人類監督範圍的權限；因 AI 決策或自動化造成的所有影響由使用者自負。
- English: The project has no moral/legal/social judgment. Do not grant AI authority beyond legal/organizational/human oversight. All consequences from AI decisions/automation are the user’s responsibility.
